<!-- SETUP -->
### SETUP

1. CLONE THE REPO
```sh
git clone https://github.com/Tymec/SortingVisualizer.git
cd SortingVisualizer
```
2. INSTALL NPM PACKAGES
```sh
yarn install
```
3. RUN THE PROJECTS
```JS
yarn dev
```
